# Overview
A mobile app designed to help families living apart (especially those with elderly parents) share updates on their loved ones' well-being and daily routines in a low-effort, flexible way.

# Usage Scenarios
Single individuals, elderly couples living alone, employees working away from home, and the “wait-and-see” period immediately after hospital discharge, etc.s

# Value Proposition
- Visualizing reassurance (that “everything is normal”)
- Early detection of changes ➡︎ Triggers conversation among family members
- Easy to maintain with a relaxed routine that isn't too demanding
